贡献代码
====================

你发现了bug或者优化了代码并且想合并到主分支？

首先你需要fork代码到你的仓库，然后切换到dev分支，在dev分支上开发完成后再github中提交 pull request，合并到dev分支。

.. note::

    只接受合并到dev分支的pull request
